## Back-End

The banking website will be hosted via Heroku which allows us to deploy the website. We will also be using MongoDB for our database management. 

## Front-End

We will also make use of React to reuse UI components and Node. js in our development due to its scalability and speed. 

## Developing Tools

The website will be developed on Visual Studio Code, intellij idea and Discord for communication as well as other tools that each developer might prefer in their part of the development stages, such as NPM, Chrome Dev tools, Mockplus classic
