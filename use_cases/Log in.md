## Log in [by Mohammad Reza]

### Actor (User)
User with a valid account.

### Pre-conditions
User must have an account

### Main Flow
1. the user clicks on log in button.
2. the system shows fields for user to enter their username/acct number and password
3. the user inputs their username/acct number and password.
4. the system will show and give access to the user into their account.  

### Alternate Flow
- if the user has forgotten their login info
  - they will have to email the bank to get a reset temporary password.

### Post Conditions
The user can successfully view all things related to their account and can use the online banking system.
